 Iad Chat
 ====================================================================================================================================================================================
 
 Prerequisites
 -------------
 
 - [Docker][2]
 - [Docker compose][3]
 - [Make][4]
 
 Installation
 ----------------
 ```
 make install
 ```
 
 Create database
  > docker exec -it 'container name' bash
  > connect to mysql 
  > execute iad_chat.sql

 The chat will be accessible on http://localhost:8000
